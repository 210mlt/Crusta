Features

Check out the list below for an overview of Crusta's features

  • Do Not Track support
  • Ability to change user agent
  • Cookie Management
  • History, Bookmarks and Download Managers
  • Custom search engine
  • Clear browsing data
  • Speed dial
  • Simple Adblock
  • Tab muting
  • Built in PDF Viewer
  • Integrated Color Picker

About

Crusta is an open-source web-browser build using QtWebEngine. Crusta aims to be light-weight privacy focussed web-browser for major desktop platforms.
